My Chess Map
========================================

> [(back to the map)](index)

This is my personal map of opening positions.
(If you're somehow here to prep against me, go away!)

Each position has its own file where I can keep notes about it.
Files are named using algebraic notation,
where spaces are replaced with underscores and without any move-qualifiers.
For example, the filename for a scholar's mate position would be:

	1.e4_e5_2.Bc4_Nc3_3.Qh5_Nf3

... followed by `.md` or `.html` depending on how you are viewing this.
Notice that Nf3 is not followed by `??`, for instance.

Each position will have a list of possible/common/notable moves
with links to the positions that result from them.
That should make navigation easy.

Transpositions are awkward, as always.
That position will just have to have some original, arbitrary move order
that it is filed under.
A bit messy, but what can you do?

Qualifiers I might use (outside of filenames ofc):

For myself, I will also denote moves with the following:

	*   I plan to play this move.
	**  I want to learn to play this move.
	*** I need to learn more about this move.

Standard algebraic qualifiers which aren't in ascii:

	++-- decisive advantage |eval| >  1.5
	--++
	+-  clear advantange   |eval| <= 1.5
	-+
	+=  small advantage    |eval| <= 0.7
	=+
	=   basically equal    |eval| <= 0.3

	[]  "only move" box
	w   "with idea" triangle
	i   infinity symbol
